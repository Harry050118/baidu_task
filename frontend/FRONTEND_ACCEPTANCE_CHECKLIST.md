# Day 11 首页验收清单

## 首页基础

- [ ] 首页可打开，路由 `/` 正常渲染指挥地图首页。
- [ ] 顶部清楚显示后端健康状态，来源为 `GET /health`。
- [ ] 首页有 loading、empty、error 状态；接口慢或失败时不白屏。
- [ ] 顶部或态势面板显示最新观测时间。
- [ ] 首页显示历史快照标识或数据新鲜度状态。

## 数据闭环

- [ ] 首页自动加载 `GET /api/map/points`。
- [ ] 首页自动加载 `GET /api/assessments`。
- [ ] 首页自动加载 `GET /api/stats/overview`。
- [ ] 风险摘要来自 `/api/assessments.items` 前端聚合。
- [ ] `/api/stats/overview` 只用于点位数、记录数、最新观测时间、坐标状态和数据新鲜度等事实信息。

## 地图与坐标

- [ ] 有坐标点可见，仅渲染 `has_coordinates=true` 且审核可用的点位。
- [ ] 缺坐标点不渲染、不编造坐标。
- [ ] 首页明确展示有坐标点数量。
- [ ] 首页明确展示缺坐标点数量。
- [ ] Marker 能表达风险等级。
- [ ] 点击可渲染点位打开右侧抽屉。

## 点位抽屉

- [ ] 抽屉展示站名。
- [ ] 抽屉展示站码。
- [ ] 抽屉展示当前水位。
- [ ] 抽屉展示风险等级。
- [ ] 抽屉展示趋势。
- [ ] 抽屉展示观测时间。
- [ ] 抽屉展示坐标状态。
- [ ] 抽屉展示坐标来源。
- [ ] “查看历史曲线”跳转 `/points/:stationCode` 可用。
- [ ] “查看研判”跳转 `/assessments` 可用。

## 高风险列表

- [ ] 高风险列表展示 `danger`、`warning`、`attention` 优先点位。
- [ ] 列表项包含风险、站名、水位、趋势、观测时间。
- [ ] 点击列表项可选中地图点位；缺少可渲染坐标时跳转点位详情。
- [ ] 无高风险时显示专业空态，不隐藏区域。
- [ ] 当前全是 `normal` 时页面仍合理展示风险摘要和空态。

## 异常与边界

- [ ] 接口失败不白屏，并显示明确错误提示。
- [ ] `/api/map/districts` 未接入。
- [ ] 未接入预测接口。
- [ ] 未接入降雨功能。
- [ ] 未接入机器学习功能。
- [ ] 前端未写入高德 Key。

## Day 12 点位详情 / 历史曲线

- [ ] 路由 `/points/:stationCode` 可打开，点位不存在时显示 404 友好文案。
- [ ] 当前水位、观测时间、风险等级、趋势提示正常显示。
- [ ] 已审核坐标显示“已审核”，并显示经纬度、坐标来源和坐标质量。
- [ ] 未审核或缺失坐标显示“坐标缺失”，不误落图、不编造坐标。
- [ ] 历史曲线档位显示为“最近 100/500/1000/5000 条”。
- [ ] 历史记录为空时显示空态，不白屏。
- [ ] 历史接口 422 时显示“请求参数不合法”或参数范围提示。

## Day 12 数据状态页

- [ ] 路由 `/data-status` 可打开。
- [ ] 积涝点数据范围显示最早时间、最新时间和记录总数。
- [ ] 最近导入批次显示导入时间、导入记录数和总行数。
- [ ] 数据新鲜度显示历史快照说明。
- [ ] 水库数据明确标注“仅状态摘要，不参与地图和积水风险主线”。
- [ ] 坐标缺失、候选、已审核数量不显示 `undefined`。
- [ ] loading / empty / error 状态清楚。

## Day 12 坐标校核页

- [ ] 路由 `/locations` 可打开。
- [ ] 左侧显示待校核积涝点列表。
- [ ] 无候选时可触发“生成高德候选坐标”。
- [ ] 候选坐标列表显示经纬度、来源和审核状态。
- [ ] 审核通过流程可用，审核后该坐标进入首页点位。
- [ ] 审核拒绝流程可用，拒绝坐标不进入首页点位。
- [ ] 高德 Key 未配置返回 503 时显示“高德服务未配置，请联系管理员”。
- [ ] 审核通过后首页 `/` 和详情页 `/points/:stationCode` 坐标状态一致。

## Day 12 规则研判页

- [ ] 路由 `/assessments` 可打开。
- [ ] 全市研判表格展示站名、水位、风险、趋势、规则说明和生成时间。
- [ ] 风险筛选可用。
- [ ] 趋势筛选可用。
- [ ] 风险与趋势筛选可组合。
- [ ] 规则说明完整展示，不截断关键内容。
- [ ] 从点位抽屉跳转 `/assessments?stationCode=...` 后目标行高亮并滚动到可见区域。
- [ ] loading / empty / error 状态清楚。

## Day 12 首页回归

- [ ] 首页 Day 11 的地图散点、点位抽屉、风险摘要、高风险列表不回退。
- [ ] 首页仍只渲染已审核坐标点。
- [ ] 首页不调用 `/api/map/districts`。
- [ ] 首页不接入预测、降雨或机器学习能力。
- [ ] 数据时间窗口不因字段名差异显示为空。
